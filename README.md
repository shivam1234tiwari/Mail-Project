This project is an email-sending service built using Node.js, Express, and Nodemailer. It provides a simple REST API to send emails and can be easily integrated into other applications.

1.Features:-

Send emails using Nodemailer

RESTful API built with Express

Environment variable support using .env

Easy to configure and extend

Suitable for learning and production use

2.Technologies Used:-

Node.js
Express.js
Nodemailer
dotenv 

How to run this project:
npm run start
